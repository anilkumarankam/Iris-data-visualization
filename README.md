# 🌸 Iris Data Visualization

A complete Data Visualization project using the famous Iris dataset. This project demonstrates how to explore and visualize data using Python libraries to identify patterns, relationships, and distributions among different Iris flower species.

---

# 📌 Project Overview

The Iris dataset is one of the most popular datasets in Data Science and Machine Learning. It contains measurements of Iris flowers from three different species.

This project focuses on creating meaningful visualizations to understand the dataset before applying machine learning algorithms.

# 📊 Dataset Information

The dataset contains **150 observations** and **5 columns**.

| Column | Description |
|---------|-------------|
| Sepal Length | Length of the sepal (cm) |
| Sepal Width | Width of the sepal (cm) |
| Petal Length | Length of the petal (cm) |
| Petal Width | Width of the petal (cm) |
| Species | Iris flower species |

There are three flower species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica
